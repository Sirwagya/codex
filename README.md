# Sirwagya Portfolio

A raw brutalist portfolio website built with HTML, CSS, and Vanilla JS.

## Features

- **Raw Brutalist Aesthetic:** Bold typography, thick borders, grayscale.
- **Smooth Scrolling:** Powered by Lenis.
- **Dynamic Projects:** Fetches from GitHub API.
- **Micro-humor:** Witty interactions.

## Setup

1. Open `index.html` in your browser to view the site.
2. To run locally with a server (recommended for API calls):
   ```bash
   npx serve .
   ```

## Customization

- **GitHub:** Update the `username` variable in `script.js` to fetch your repositories.
- **Spotify:** Replace the mock in `script.js` with a real Spotify Embed iframe.
- **Contact Form:** Point the form action to a service like Formspree or Netlify Forms.

## Deployment

This is a static site. You can deploy it to:

- GitHub Pages
- Netlify
- Vercel
